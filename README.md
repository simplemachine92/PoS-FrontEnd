# 🏄‍♂️ Quick Start

> For Proof of Stake:

Install,

```bash
cd greenpill
git checkout steak-style-updates
yarn install
```

local node, deploy, and launch react app.

```bash
yarn chain
yarn deploy
yarn start
```
